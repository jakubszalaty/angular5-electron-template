# Angular 5 Electron Template

## Setup

```sh
yarn install
# or
npm install
```


## Start development

Run dev server
```sh
yarn dev
# or
npm run dev
```

## Run in production


```sh
yarn start
# or
npm run start
```

## Build execute file

```sh
yarn build:electron:linux
yarn build:electron:win
# or
npm run build:electron:linux
npm run build:electron:win
```

Files are stored in `build` folder.
